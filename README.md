# Calculadora Python

## Objetivo
Projeto desenvolvido para a atividade prática de Git e GitHub, aplicando conceitos de versionamento, branches, commits, issues e pull requests.

## Funcionalidades
- Soma de dois números
- Subtração de dois números
- Multiplicação de dois números
- Divisão de dois números com validação de divisão por zero
- Menu interativo no terminal
- Validação de entrada do usuário

## Como executar
1. Ter Python instalado
2. Rodar no terminal: `python app.py`
3. Escolher a operação desejada e digitar os números

## Tecnologias
- Python 3
- Git
- GitHub
