# Calculadora Financeira e Administrativa Python

## Objetivo
Projeto desenvolvido para a atividade prática de Git e GitHub, aplicando conceitos de versionamento, branches, commits, issues e pull requests. Ferramenta de console que engloba operações aritméticas e cálculos financeiros usados em administração.

## Funcionalidades
- Operações básicas: soma, subtração, multiplicação, divisão
- Juros simples: cálculo de juros e montante
- Juros compostos: cálculo de montante composto
- Valor presente (desconto composto)
- Valor futuro e presente de anuidades
- Cálculo de parcela de empréstimo (PMT)
- Tabelas de amortização: Sistema Price e SAC
- Conversão entre taxa nominal e taxa efetiva
- Reajuste por inflação
- Retorno sobre investimento (ROI)
- Menu interativo no terminal com validação de entrada

## Como executar
1. Ter Python instalado (3.7+)
2. No terminal, dentro do diretório do projeto: `python app.py`
3. Escolher a operação no painel e seguir as instruções

## Exemplos de uso
- Calcular montante composto: escolha a opção 6, informe capital, taxa e períodos.
- Gerar tabela de amortização Price: escolha a opção 11, informe valor do empréstimo, taxa e número de períodos.

## Tecnologias
- Python 3
- Git
- GitHub
